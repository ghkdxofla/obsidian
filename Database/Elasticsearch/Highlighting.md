# Abstract
- 특정 Query로 검색 후 `match`(또는 `terms`)를 통해 확인된 검색 결과를 강조하는 기능
- `fragment_size`를 통해 검색된 내용의 일부분을 추출할 수 있음
- `highlight`된 결과는 `<em>highlight</em>` 으로 기본 출력됨
