# Abstract
- 컴파일러를 이용해 런타임 중에 컴파일 및 최적화
- 애플리케이션이 실행되면서 중간 언어인 바이트 코드를 기게어로 번역
- 프로파일링을 통해 얻어낸 실행 환경 정보를 바탕으로 최적화를 진행
	-  Intel CPU를 사용중이라면 Intel CPU에 맞게 바이트 코드가 기게어로 번역되면서 최적화가 되는 것
- `동적 컴파일러`
