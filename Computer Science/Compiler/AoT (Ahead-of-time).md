# Abstract
- 미리 특정 디바이스에 맞는 기계어로 컴파일을 해두는 것
- 컴파일 과정에서 최적화 진행
- AoT 컴파일러의 결과로 즉시 실행 가능한 네이티브 실행 파일이 생성
- [[JIT (Just-in-time)]] 컴파일러와 다르게 실행 시 최적화되지 않음
	- 이미 기계어로 컴파일이 되었고, 특정 환경에 맞게 최적화 되어 실행 환경 정보가 수집되어도 적용이 어려움
- 컴파일 과정에서 복잡한 분석 및 최적화 수행
- `정적 컴파일러`
